ACCÈS ADMIN CLEAN-CITÉ

URL : https://clean-cite.org/admin/

Si la connexion admin ne fonctionne pas :
1. Netlify > votre site > Identity : activer Identity.
2. Identity > Registration preferences : choisir Open pour permettre aux clients de créer leur compte librement.
3. Identity > Services > Git Gateway : Enable Git Gateway.
4. Identity > Invite users : inviter l’email admin.
5. Accepter l’invitation reçue par email et créer le mot de passe.
6. Retourner sur https://clean-cite.org/admin/
